About this Website

This website was developed on the basis of a Udemy course project. The main tech stack used in backend are Spring Boot, Thymeleaf, JPA, Mysql. In Frontend, the HTML, CSS, Semantic-UI, Javascript, jQuery and some libaries of Javascript were employed.

The admin system of this website was completely restructured. I use JWT token instead of session to make every request statless. Every time when the user login in successfully, token was stored in SessionStorage of the Browser. All the request in Admin system was made via Ajax becaus of request Header "Authorization" need to be added. Therefore, the Admin system of this blog website is more like a single page application and only a little thymeleaf are employed for convenience.
